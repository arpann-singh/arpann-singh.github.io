Campus Connect: A Lost & Found Platform

Problem & Solution

"Campus Connect" is a mobile-first application designed to be the centralized hub for lost and found items on a college campus. It streamlines the recovery process by instantly and securely connecting the person who lost an item with the person who found it, maximizing recovery chances.

Technology Stack

  • Mobile Framework: Flutter (Dart)
  • Backend: Google Firebase (Authentication, Cloud Firestore, Cloud Storage)
  • Serverless Functions: Cloud Functions for Firebase (TypeScript)
  • Notifications: Firebase Cloud Messaging (FCM)
